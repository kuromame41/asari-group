// about

体験を通じて、
記憶をつくる

あさりぐるーぷは、おぬり・ちのり・いくりの3人からなるクリエイティブ集団です。小学5年生の合宿をきっかけに謎解きゲームをつくり始め、中学・高校と活動を続けながら成長してきました。

単なるパズルではなく、ストーリーと感情が交差する「体験」にこだわって制作しています。謎解き・脱出ゲーム・イベント企画など、様々な形で依頼を受け付けています。

// history

あさりぐるーぷの軌跡

🧩

小学校｜原点と創造の芽生え

2018年・清里合宿でのグループ対抗周遊型脱出ゲームがすべての始まり。紙と鉛筆だけで、本格的な謎解き体験を作り出した時代。

📖 代表作品

  • 人狼村からの脱出
  • 消えた神のペンダント
  • 桜の花が散るときに
  • おにまめ
  • 雪下紫陽

✨ この時代のポイント

  • 友人を登場人物に入れたストーリー謎解き
  • 謎の答えが次の場所を示す周遊型構成
  • 景品でモチベーション設計
  • 地下謎・WESTCODEから影響を受ける
  • 小学5年「ENIGMA」→あさりぐるーぷの前身へ

🛠 使用ツール

紙・鉛筆（手書き） InDesign（独学）
🌀

中学校｜制限下での模索と工夫

コロナ禍と校則の壁の中、紙媒体から動画へと形式を進化させた。メタ構造・分岐エンディング・Blender演出など実験的な挑戦を続けた時代。

📺 代表作品

  • パズる！（2020・校内掲示＋LINE配信）
  • カムソミトラ（2021・観客＝主人公のメタ構造）
  • 時限爆弾が仕掛けられた学校からの脱出（2022）
  • 卒業歓送会謎解き（2023）

🎬 演出上の工夫

  • 観客＝主人公のメタ構造を初導入
  • 時間制限付き謎解きパート
  • Blenderで学校爆発シーンを再現
  • エンディング分岐形式を採用
  • 成功チームが出ないとGAME OVER構造
  • 2日目にヒント制度を追加

🛠 使用ツール

ロイロノート Googleフォーム Clipchamp AviUtl Blender Unity InDesign Canva
🚀

高校｜技術と表現力の飛躍

「印刷」の導入、脱出ゲーム甲子園への出場、本格的なチーム運営へ。「遊び」から「制作」へと変化した時代。

🏆 代表作品

  • 墜落する宇宙船からの脱出（高1桐陰祭）
  • おふとんからの脱出（脱出ゲーム甲子園出場）

🎯 甲子園作「おふとんからの脱出」

  • 会場全体を"おふとんの結界"に演出
  • 朝のミーティングで誰も来ないことが発想源
  • SCRAPの方針を意識した導線設計
  • 発表会当日最終回まで改良を続けた渾身作

⚙️ この時代の進化

  • 印刷・ラミネート加工を初導入
  • アニメーション＋会話形式の演出
  • 選抜メンバー制・分業体制の確立
  • Googleスプレッドシートで日程管理
  • 外部評価を初めて獲得

🛠 使用ツール

Unity Canva AviUtl Googleドキュメント スプレッドシート Googleフォーム
